# Godot game: Energetic

That's a new project, managed by 3 persons

We're going to create a video game with Godot:
  - 2D game
  - GDscript
  - Free customized assets

# Specifications:

  -Gameplay: Jump to flip and run backward, clone yourself, run nonstop
  -Enemies: // No enemies

The character runs nonstop and must avoid getting stuck on obstacles, falling into holes, and being killed by circular saws.
Press the spacebar to reverse gravity and make him run backward on the upper platform.
